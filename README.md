# Notes — клон Apple Notes для Windows 11

Десктоп-приложение в стиле Apple Notes, собирается в `.exe` под Windows 11 (x64).

## Что внутри

- Трёхколоночный интерфейс: папки → список заметок → редактор
- Фирменный жёлтый акцент, системный шрифт (Segoe UI Variable на Windows), светлая/тёмная тема следует системной
- Папки: создание, переименование (двойной клик), удаление (правый клик)
- Заметки: создание, удаление, **закрепление**, автосохранение (~400 мс после ввода)
- **Поиск** по заголовкам и содержимому
- **Форматирование**: жирный / курсив / подчёркивание, маркированные и нумерованные списки
- **Чек-листы** (клик по кружку отмечает пункт)
- **Экспорт в PDF** для текущей заметки
- Локальное хранение в SQLite (`%APPDATA%\notes-clone\notes.db`)

## Требования для сборки

- **Windows 11** (x64)
- **Node.js 18+** — [скачать](https://nodejs.org/)
- ~2 ГБ свободного места на диске (node_modules + сборка)

> **Важно:** `better-sqlite3` — нативный модуль. При первом `npm install` он скомпилируется под вашу версию Node.js. Для этого нужны **Visual Studio Build Tools** (компонент "Desktop development with C++"). Если их нет, установщик Node.js предложит поставить их автоматически на последнем шаге — соглашайтесь. Либо поставьте вручную: [Build Tools для VS 2022](https://visualstudio.microsoft.com/visual-cpp-build-tools/).

## Запуск в режиме разработки

```bash
npm install
npm start
```

Откроется окно приложения. Можно пользоваться, все изменения сохраняются в локальную БД.

## Сборка `.exe` под Windows 11

### Вариант 1 — установщик (рекомендуется)

```bash
npm run build:installer
```

Результат в папке `dist/`:
- `Notes Setup 1.0.0.exe` — установщик NSIS (~80 МБ). Запускаете его, выбираете папку установки, получаете ярлыки на рабочем столе и в меню «Пуск».

### Вариант 2 — портативная версия (без установки)

```bash
npm run build:portable
```

Результат:
- `dist/Notes-Portable-1.0.0.exe` — запускается напрямую, данные пишутся в `%APPDATA%\notes-clone\`.

### Вариант 3 — оба сразу

```bash
npm run build
```

## Частые вопросы

**Q: При `npm install` падает `better-sqlite3`.**
A: Установите Visual Studio Build Tools (компонент C++). После этого: `npm rebuild better-sqlite3` или удалите `node_modules` и запустите `npm install` заново.

**Q: Windows SmartScreen ругается на .exe.**
A: Это нормально для неподписанных сборок. Нажмите «Подробнее» → «Выполнить в любом случае». Чтобы убрать предупреждение, нужна покупка сертификата подписи кода (~$200-400/год).

**Q: Где хранятся мои заметки?**
A: В `%APPDATA%\notes-clone\notes.db` (SQLite-файл). Для бэкапа скопируйте этот файл.

**Q: Как перенести заметки на другой компьютер?**
A: Скопируйте `notes.db` в ту же папку на новой машине.

**Q: Можно ли изменить дизайн?**
A: Да — вся стилизация в `src/styles.css`. Основные цвета задаются CSS-переменными в `:root` и `@media (prefers-color-scheme: dark)`.
